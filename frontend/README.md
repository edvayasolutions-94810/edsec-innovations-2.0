# EDSEC Innovations Platform

A modern, MSME certified training institute platform offering certified internships in Data Science, Data Analytics, Full Stack Development, Web Development, Python & SQL.

## About

EDSEC Innovations is a certified training institute headquartered in Bengaluru, dedicated to transforming aspiring professionals into industry-ready experts through hands-on internship programs.

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
node index.js
```
